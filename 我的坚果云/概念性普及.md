#### Servlet规范

#### 什么是动态资源文件

只有实现Servlet接口的Java类才是 【动态资源文件】，只有这样的类才能被Tomcat调用。

例如：

```java
class Student{
  // 不是动态资源文件，Tomcat无权调用
}

class Teacher implements Servlet{
  // 是动态资源文件，Tomcat有权利创建和调用这个类的实例对象
  
  void doGet() {
    ....
  }
}
```

怎么请求呢？

tomcat收到请求后，有权利创建Teacher类的实例对象，然后调用Teacher类的相关方法。

```java
Servlet obj = new Teacher();
obj.doGet();
```
