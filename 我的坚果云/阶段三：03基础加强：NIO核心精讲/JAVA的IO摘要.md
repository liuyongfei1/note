### JAVA IO中常用的类

| 类               | 说明           |
| ---------------- | -------------- |
| File             | 文件类         |
| RandomAccessFile | 随机存取文件类 |
| InputStream      | 字节输入流     |
| OutPutStream     | 字节输出流     |
| Reader           | 字符输入流     |
| Writer           | 字符输出流     |

在整个java.io包中：

- 最重要的5个类，分别是File，InputStream，OutPutStream，Reader，Writer；
- 一个接口指的是：Serializable。
